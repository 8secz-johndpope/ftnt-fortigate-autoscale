# FortiGate Autoscale for Azure deployment package

This folder contains a template for deploying FortiGate Autoscale for Azure. The button below can be used to start the deployment. For details regarding parameters as well as activities that must occur before and after deploying the templates, refer to the instructions in the project [README](https://github.com/fortinet/fortigate-autoscale/blob/master/README.md).
